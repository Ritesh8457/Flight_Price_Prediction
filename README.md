# Flight_Price_Prediction
🚀 Completed My Flight Price Prediction Machine Learning Project ✈️📊

I recently completed a Flight Price Prediction project using Python and Machine Learning, where I analyzed airline ticket data and built predictive models to estimate flight prices.

🔹 What I implemented in this project:
✅ Data Cleaning & Preprocessing
✅ Handling Missing Values and Duplicate Data
✅ Exploratory Data Analysis (EDA) using Matplotlib & Seaborn
✅ Feature Engineering for Date, Time, Duration & Stops
✅ Correlation Analysis & Feature Importance
✅ Model Building using:

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor

🔹 Problem I solved:
The dataset contained missing values, duplicate records, and categorical features that could not be directly used for machine learning models.
I solved this by:
✔️ Cleaning the dataset
✔️ Encoding categorical variables
✔️ Extracting useful features from date and time columns
✔️ Applying model tuning to improve prediction accuracy

🔹 Final Result:
After comparing multiple models, the Tuned Random Forest Regressor achieved the best performance for predicting flight ticket prices.

Python | Pandas | NumPy | Matplotlib | Seaborn | Scikit-learn
